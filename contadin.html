Contadin

Es un programa para ayudarte con las cuentas basado en la contabilidad de doble asiento.
Su interfaz permite la utilización por personas no familiarizadas con términos contables. (al menos eso se intenta)
Se estructura en cuatro tipos de Cuentas:
INGRESOS
GASTOS
PATRIMONIO
DEUDAS
Como actualmente todos recibimos las facturas, los apuntes del banco, etc. en formato digital, las últimas versiones incorporan la opción de archivarlos de forma ordenada y vinculados a los apuntes de nuestra contabilidad.

Contadin Access

Se ejecuta bajo sistema operativo Windows:
XP o Superior (32 y 64 bits)
Desarrollado bajo la arquitectura de Access: (Microsoft-Office)en cualquiera de las versiones: 2003, 2007, 2010
Los usuarios que no tienen instalado Access deben instalar previamente AccessRunTime.
AccessRunTime es una aplicación gratuíta de Microsoft, a la que puedes acceder en la pestaña ENLACES

Contadin Visual Studio

Se ejecuta bajo sistema operativo Windows:
XP o Superior (32 y 64 bits)
Desarrollado bajo la arquitectura .NET Framework 4.0 de Microsoft con el Software de Visual Studio 2010
Los actuales sistemas operativos Windows incorporan Framework 4.0 o superior.
Si Framework 4.0 no está presente en la máquina del usuario la instalación del programa te dirige a la página de Microsoft para su descarga gratuíta.
