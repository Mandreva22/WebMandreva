Crear el instalador de una Aplicación

 

Nuestro objetivo es crear un solo archivo (que puede tener la extensión .exe o bien .msi) que instale un programa en la máquina del usuario. 

 

Paso previo es tener preparados los archivos que queremos distribuir con el instalador. En este ejemplo queremos instalar la base de datos “Contadin V201503.mdb” en la máquina del usuario y crear un acceso directo en el escritorio y en el menú programas.

 

En  este caso los archivos son:

CONTADIN V201406.mdb

Distribución Usuarios Informacion.rtf

Distribución Usuarios Licencia.rtf

También queremos personalizar los accesos directos y los mensajes del instalador con nuestro logo y utilizamos los siguientes archivos:

Contadin.ico

Instalador1.bmp

Instalador2.jpg

Ahora debemos asegurarnos que el usuario tiene en su máquina los archivos necesarios para que el Instalador funcione.

Así que nos en el Explorador de Soluciones nos situamos sobre nuestro proyecto y con el botón de derecho, en el menú contextual, seleccionamos Propiedades. Se abrirá esta ventana:


Pulsamos sobre ‘Requisito Previos…’ y tenemos esta pantalla:


Aquí tenemos que asegurarnos que está habilitada la opción inicial de ‘Crear programa de instalación para instalar los componentes necesarios’, y también Descargar los requisitos previos del sitio Web del proveedor de los componentes.

Luego dentro de Elegir los requisitos previos que se van a instalar como mínimo tienen que estar habilitados estos dos:

Microsoft.NET Framework 4 Client Profile (x86 y x64)

Windows Installer 3.1

 

En el caso improbable de que el usuario no tenga instalados estos dos requisitos el instalador le lanzarán un mensaje para redirigirlo a la web de descargas de Microsoft

No te olvides en cada paso de ir guardando el proyecto.
